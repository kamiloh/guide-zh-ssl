# guide-zh-ssl

本仓库用于归档和发布多个独立的 HTML 页面，内容主要围绕中文环境下的 SSL/TLS 相关主题。页面均为静态单页，不依赖后端服务，可直接在浏览器中打开或托管于任何静态文件服务。

## 项目简介

这是一个以 SSL 技术资料整理为目的的仓库，收录了多份独立编写的 HTML 文档。每份文档侧重于不同侧面，例如配置示例、概念解释、常见问题梳理等。所有页面均以中文编写，旨在为开发者、运维人员或感兴趣的用户提供一份可离线查阅的参考集合。

## 目录说明

```
guide-zh-ssl/
├── README.md              # 本文件
├── index.html             # 仓库入口页（可选）
├── pages/                 # 存放所有独立 HTML 页面
│   ├── ssl-basics.html    # SSL/TLS 基础概念
│   ├── config-example.html # 配置示例与说明
│   ├── faq.html           # 常见问题汇总
│   └── ...                # 其他页面
└── assets/                # 静态资源（CSS、图片等）
    ├── style.css
    └── images/
```

- `pages/` 目录下每个文件均为完整的独立 HTML 页面，可直接访问。
- `assets/` 目录存放页面共用的样式文件与图片资源。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，无需额外环境即可浏览。
- 页面之间没有交叉引用或依赖关系，可单独使用。
- 内容更新时，会以新版本覆盖原文件或在 `pages/` 下新增文件。
- 不针对任何特定域名或网站，所有示例均使用通用占位符。
